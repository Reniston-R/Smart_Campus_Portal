# 🚌 CampusTransit — Smart Campus Transport Portal

A modern, real-time campus bus management system designed to reduce delays, streamline routes, and keep students moving seamlessly.

---

## 🌟 Features

- **🏠 Home Dashboard** — Live stats including active buses, campus stops, average delay, and daily riders
- **🕒 Bus Timetable** — Today's scheduled departures with real-time on-time / delayed / upcoming statuses
- **🗺️ Route Map** — Visual campus bus network showing all stops with estimated travel times
- **⏱️ Travel Time Calculator** — Estimate journey time based on distance and bus speed
- **📊 Student Demand** — Peak hour analysis and hourly ridership bar chart

---

## 🛠️ Tech Stack

- **HTML5** — Semantic structure
- **CSS3** — Custom dark-mode glassmorphism design system with animations
- **Vanilla JavaScript** — Panel navigation, calculator logic

---

## 🚀 Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/Reniston-R/Smart_Campus_Portal.git
   ```

2. Open `index.html` in your browser — no build step required!

- **Dark mode** with deep navy background (`#0f172a`)
- **Glassmorphism** cards with `backdrop-filter: blur`

- **Smooth panel transitions** with fade-in animations
